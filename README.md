# The Goal: RacingKings.org

Inspired by the **Lichess philosophy**, building a dedicated, Free Forever, Non-Profit Open Source **home for the Racing Kings chess variant community.**

## Status:

This project is in its very early infancy. We are currently architecting the core bitboard engine and the FFI bridge. Expect bugs, expect frequent changes, but most importantly—expect speed.

## Credits & Legal

* **Core Inspiration:** Part of Core bitboard logic adapted from [Stockfish](https://github.com/official-stockfish/Stockfish).
* **Philosophy & Variant Rules:** Inspired by [Lichess.org](https://github.com/lichess-org) and their commitment to free, open-source chess.
* **Variant Research:** Move generation logic references from the [Fairy-Stockfish](https://github.com/ianfab/Fairy-Stockfish) project.
